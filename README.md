# Para para clonar el repositorio

1.Iniciar un repositorio de visual studio code

2.Realizar el comando git .init para iniciar el repositorio

3. Utilizar el comando git clone junto a la URL mediante HTTPS: https://github.com/ElxCoNe/prueba-html-riwi.git

4.Ingresar a la carpeta que se crea automaticamente

5. Encontrarás todos los recursos utilizados para llevar a cabo este proyecto
